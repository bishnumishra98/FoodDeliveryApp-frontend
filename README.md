# Food Delivery Platform (MERN)

A full-stack food delivery application supporting customer ordering, real-time order tracking, admin management, and payment integration.

## Tech Stack
- Frontend: React.js, Redux, Socket.io
- Backend: Node.js, Express.js, MongoDB
- Auth: JWT, bcrypt
- Media: Multer, Cloudinary
- Payments: PhonePe (Sandbox)
- Deployment: Netlify (Frontend), Render (Backend)

## Key Features
- User authentication and role-based access (Customer / Admin)
- Restaurant menu browsing and cart management
- Real-time order status updates using WebSockets
- Secure payment flow using PhonePe test environment
- Admin dashboard for managing items and orders

## Architecture Overview
- REST-based backend with JWT-secured APIs
- WebSocket layer for real-time updates
- Stateless backend services with MongoDB persistence
